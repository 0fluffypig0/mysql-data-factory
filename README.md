# MySQL Data Factory

MySQL批量数据操作工具 - 支持离线环境部署

##  功能特性

- ✅ 离线环境部署（支持无网络堡垒机）
- ✅ 批量数据插入（高性能）
- ✅ 样本驱动的数据生成（从真实记录生成测试数据）
- ✅ Jupyter Notebook交互式操作
- ✅ 数据模板加载（CSV/Excel/JSON）
- ✅ 完整的日志系统

## 🚀 快速开始

### 1. 配置数据库连接

复制 `.env.example` 为 `.env`，并修改为你的数据库信息：

```bash
cp .env.example .env
